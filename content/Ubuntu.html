Ubuntu

Ubuntu LTS Life Cycle

https://ubuntu.com/about/release-cycle

安裝注意事項:

  1. 當採用純 IPv6 安裝 Ubuntu, 並在安裝過程設定代理主機後, 系統會將設定存入 /etc/apt/atp.conf.d 目錄中的 90curtin-aptproxy 檔案中, 若之後不需要在執行 apt 指令時使用 Proxy 代理主機時, 必須刪除檔案中的設定.

  2. 安裝 fcitx 中文輸入

    step 1

    sudo apt install fcitx fcitx-chewing

    step 2

    sudo mv /usr/bin/ibus-daemon /usr/bin/ibus-daemon.bak
    sudo reboot

    step 3

    右上方以滑鼠點選 input method 中的 Configure 進入 Input Method Configuration 視窗設定畫面.
    畫面將只呈現 Keybord - English (US)
    必須增加 Chewing input method, 方法為點選左下方的 + 號, 去除 Only Show Current Language 選項, 然後用搜尋, 輸入 Chewing 表示要將注音輸入法納入 Input Method 中.

    完成上述三項步驟後, 就可以利用 Ctrl + Space 或鍵盤左側的 Shift 鍵切換中英輸入, 選擇同音字時則利用方向鍵與數字鍵進行選字.

    fcitx 中文輸入是依照帳號設定

    安裝後, 必須逐一為每一個帳號設定使用 fcitx 中文輸入法, 而且 input method 中的 configure 中的 Addon 中的 Simplified Chinese To Traditional Chinese/Convert Simplified Chinese To Traditional Chinese 選項若打勾, 將會使用簡體中文, 若取消選擇, 才會使用繁體中文輸入系統.

  3.  create admin user

    $sudo adduser username
    $sudo usermod -aG sudo username

  4. correct apt upgrade error

    While prepare cd12 for cad2019 course server, same error comes up: https://www.ostechnix.com/fix-dpkg-error-parsing-file-var-lib-dpkg-updates-0014-error-in-ubuntu/

    $ sudo rm /var/lib/dpkg/updates/0015
    $ sudo apt clean
    $ sudo apt update
    $ sudo dpkg --configure -a
    $ sudo apt upgrade

    before install nginx, uwsgi, squid and fossil scm, Ubuntu need to make sure all the system packages is the most updated version.

  5. install squid

    sudo apt install squid
    cd /etc/squid
    sudo rename squid.conf squid_orig.conf
    cd
    sftp username@ipv4_of_another_server
    cd /etc/squid
    get squid.conf

    quit
    sudo cp squid.conf /etc/squid
    sudo /etc/init.d/squid restart

    change hostname
    sudo hostnamectl set-hostname your_server_hostname
    sudo vi /etc/hosts
    add 127.0.0.1 your_server_hostname

  6. reset Ubuntu password
    while booting, for bios startup computer, press shift, for uefi start computer, press esc to go to the boot menu, select from menu to boot for network and root setup, use sudo passwd username to change user password.
